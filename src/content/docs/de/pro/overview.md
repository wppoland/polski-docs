Polski PRO for WooCommerce ist eine Premium-Erweiterung zum kostenlosen Plugin Polski for WooCommerce. Es fuegt erweiterte Module fuer polnische Online-Shops hinzu - von der Rechnungsstellung und KSeF-Integration ueber den mehrstufigen Checkout bis hin zu Geschenkkarten, Abonnements und einem Partnerprogramm.

## Was die PRO-Version bietet

Die kostenlose Version von Polski for WooCommerce deckt die grundlegenden Anforderungen polnischer Shops ab: NIP-Felder, rechtliche Checkboxen, Stueckpreise, MwSt.-Anzeige und Omnibus-Konformitaet. Die PRO-Version erweitert diese Moeglichkeiten um acht zusaetzliche Module:

| Modul | Beschreibung |
|-------|------|
| [Rechnungen](/pro/invoices/) | System fuer MwSt.-Rechnungen, Korrekturrechnungen, Quittungen und Lieferscheine mit PDF-Generierung |
| [KSeF-Integration](/pro/ksef/) | Versand elektronischer Rechnungen an das Nationale e-Rechnungssystem (KSeF) |
| [Mehrstufiger Checkout](/pro/multistep-checkout/) | Aufteilung des Bestellvorgangs in Schritte: Adresse, Versand, Zahlung, Zusammenfassung |
| [Einwilligungsverwaltung](/pro/consent-management/) | Versionierung von Einwilligungen, Audit-Trail, CSV/JSON-Export, DSGVO-Integration |
| [Geschenkkarten](/pro/gift-cards/) | Verkauf von Geschenkkarten, Code-Generierung, Einloesung im Warenkorb |
| [Abonnements](/pro/subscriptions/) | Produkte mit wiederkehrender Zahlung, Verlaengerungen, E-Mail-Erinnerungen |
| [Partnerprogramm](/pro/affiliates/) | Empfehlungslinks, Provisionsverfolgung, Partner-Dashboard |

Jedes Modul kann in den Plugin-Einstellungen unabhaengig aktiviert oder deaktiviert werden.

## Systemanforderungen

| Anforderung | Mindestversion |
|-----------|-----------------|
| Polski for WooCommerce (free) | 1.3.0 |
| WordPress | 6.4 |
| WooCommerce | 8.0 |
| PHP | 8.1 |

Das PRO-Plugin funktioniert als Erweiterung zur kostenlosen Version - es erfordert deren Vorhandensein und Aktivierung. Stellen Sie vor der Installation von PRO sicher, dass die kostenlose Version installiert und auf mindestens Version 1.3.0 aktualisiert ist.

## Installation

### Schritt 1 - ZIP-Datei herunterladen

Nach dem Kauf der Lizenz auf [wppoland.com/polski-pro](https://wppoland.com/polski-pro) laden Sie die ZIP-Datei des Plugins aus dem Kundenbereich herunter.

### Schritt 2 - Plugin installieren

1. Gehen Sie zu **Plugins > Installieren > Plugin hochladen**
2. Waehlen Sie die heruntergeladene ZIP-Datei aus
3. Klicken Sie auf **Jetzt installieren**
4. Klicken Sie nach der Installation auf **Plugin aktivieren**

### Schritt 3 - Lizenz aktivieren

1. Gehen Sie zu **WooCommerce > Einstellungen > Polski > PRO-Lizenz**
2. Geben Sie den nach dem Kauf erhaltenen Lizenzschluessel ein
3. Klicken Sie auf **Lizenz aktivieren**
4. Nach erfolgreicher Aktivierung erscheint eine gruene Meldung mit dem Ablaufdatum der Lizenz

Der Lizenzschluessel ist an die Shop-Domain gebunden. Wenn Sie Ihren Shop auf eine andere Domain umziehen, deaktivieren Sie zuerst die Lizenz auf der alten Domain und aktivieren Sie sie dann auf der neuen.

### Schritt 4 - Module aktivieren

Gehen Sie nach der Lizenzaktivierung zu **WooCommerce > Einstellungen > Polski > PRO-Module** und aktivieren Sie die gewuenschten Module. Jedes Modul hat einen eigenen Konfigurationsbereich, der in der jeweiligen Dokumentation beschrieben ist.

## Aktualisierungen

Das PRO-Plugin unterstuetzt automatische Aktualisierungen ueber das WordPress-Dashboard. Nach der Lizenzaktivierung werden Aktualisierungen genauso bereitgestellt wie bei Plugins aus dem WordPress.org-Repository:

1. Eine Benachrichtigung ueber eine verfuegbare Aktualisierung erscheint im Bereich **Plugins**
2. Klicken Sie auf **Jetzt aktualisieren**
3. Das Plugin wird automatisch aktualisiert

Aktualisierungen erfordern eine aktive Lizenz. Nach Ablauf der Lizenz funktioniert das Plugin weiterhin, erhaelt jedoch keine Aktualisierungen mehr.
