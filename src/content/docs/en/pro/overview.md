Polski PRO for WooCommerce is a premium extension for the free Polski for WooCommerce plugin. It adds advanced modules for Polish online stores - from invoicing and KSeF integration, through multi-step checkout, to gift cards, subscriptions and an affiliate program.

## What the PRO version adds

The free version of Polski for WooCommerce covers the basic requirements of Polish stores: VAT ID fields, legal checkboxes, unit prices, VAT display and Omnibus compliance. The PRO version extends these capabilities with eight additional modules:

| Module | Description |
|--------|-------------|
| [Invoices](/pro/invoices/) | VAT invoice system, corrections, receipts and packing slips with PDF generation |
| [KSeF integration](/pro/ksef/) | Sending electronic invoices to the National e-Invoice System |
| [Multi-step checkout](/pro/multistep-checkout/) | Splitting the order process into steps: address, shipping, payment, summary |
| [Consent management](/pro/consent-management/) | Consent versioning, audit trail, CSV/JSON export, GDPR integration |
| [Gift cards](/pro/gift-cards/) | Selling gift cards, code generation, redemption in cart |
| [Subscriptions](/pro/subscriptions/) | Products with recurring payments, renewals, email reminders |
| [Affiliate program](/pro/affiliates/) | Referral links, commission tracking, affiliate dashboard |

Each module can be enabled or disabled independently in the plugin settings.

## System requirements

| Requirement | Minimum version |
|-------------|----------------|
| Polski for WooCommerce (free) | 1.3.0 |
| WordPress | 6.4 |
| WooCommerce | 8.0 |
| PHP | 8.1 |

The PRO plugin works as an add-on to the free version - it requires it to be present and active. Before installing PRO, make sure the free version is installed and updated to at least version 1.3.0.

## Installation

### Step 1 - download the ZIP file

After purchasing a license at [wppoland.com/polski-pro](https://wppoland.com/polski-pro), download the plugin ZIP file from the customer dashboard.

### Step 2 - install the plugin

1. Go to **Plugins > Add New > Upload Plugin**
2. Select the downloaded ZIP file
3. Click **Install Now**
4. After installation, click **Activate Plugin**

### Step 3 - activate the license

1. Go to **WooCommerce > Settings > Polski > PRO License**
2. Enter the license key received after purchase
3. Click **Activate License**
4. After successful activation, a green message with the license expiration date will appear

The license key is tied to the store domain. If you are moving your store to a different domain, first deactivate the license on the old domain, then activate it on the new one.

### Step 4 - enable modules

After activating the license, go to **WooCommerce > Settings > Polski > PRO Modules** and enable the desired modules. Each module has its own configuration section described in the dedicated documentation.

## Updates

The PRO plugin supports automatic updates through the WordPress dashboard. After license activation, updates are delivered the same way as for plugins from the WordPress.org repository:

1. A notification about an available update will appear in the **Plugins** dashboard
2. Click **Update Now**
3. The plugin will update automatically

Updates require an active license. After the license expires, the plugin continues to work but does not receive updates.
