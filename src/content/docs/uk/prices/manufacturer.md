З 13 грудня 2024 р. регламент GPSR вимагає вказувати дані виробника на сторінці продукту. Плагін Polski for WooCommerce дозволяє додати дані виробника, бренд та номер GTIN/EAN до кожного продукту.

## Вимоги GPSR

На сторінці продукту потрібно вказати:

- назва виробника або імпортера
- поштова адреса виробника
- електронна адреса або вебсайт для зв'язку
- у випадку продуктів не з ЄС - дані відповідальної особи на території ЄС

Ця інформація повинна бути легко доступною для споживача перед покупкою.

## Налаштування

### Увімкнення модуля

Перейдіть до **WooCommerce > Налаштування > Polski > Виробник** та активуйте модуль. Після активації в редакторі продукту з'являться нові поля.

### Дані виробника (GPSR)

У редакторі продукту, у вкладці "Polski" або в бічній панелі, ви знайдете секцію "Виробник (GPSR)":

| Поле | Обов'язкове | Опис |
|------|-------------|------|
| Назва виробника | Так | Повна назва компанії виробника |
| Адреса | Так | Вулиця, номер, поштовий індекс, місто, країна |
| E-mail | Так* | Контактна електронна адреса |
| Вебсайт | Так* | URL сайту виробника |
| Відповідальна особа в ЄС | Умовно | Обов'язково для продуктів не з ЄС |
| Адреса відповідальної особи | Умовно | Повна адреса відповідальної особи |

*Необхідний щонайменше один спосіб електронного зв'язку (e-mail або вебсайт).

### Глобальні дані виробника

Якщо ви продаєте переважно продукцію власного бренду, ви можете встановити дані виробника за замовчуванням у **WooCommerce > Налаштування > Polski > Виробник**. Ці дані автоматично застосовуватимуться до всіх продуктів, які не мають призначених індивідуальних даних виробника.

## Таксономія бренду

Плагін реєструє таксономію `polski_brand`, що дозволяє керувати брендами продуктів.

### Управління брендами

Перейдіть до **Продукти > Бренди**, щоб створювати та редагувати бренди. Кожен бренд може містити:

- назву
- slug (ідентифікатор URL)
- опис
- логотип (мініатюра таксономії)

### Призначення бренду продукту

У редакторі продукту, в бічній панелі, ви знайдете метабокс "Бренд" - виберіть бренд зі списку або додайте новий.

### Сторінки бренду

Плагін автоматично генерує сторінки архіву для кожного бренду. Клієнти можуть переглядати всі продукти певного бренду за адресою:

```
/marka/nazwa-marki/
```

Slug архіву можна змінити в налаштуваннях плагіна.

## GTIN/EAN

Плагін додає поле для ідентифікаційного номера продукту, сумісного зі стандартами GS1.

### Підтримувані формати

| Формат | Довжина | Застосування |
|--------|---------|-------------|
| EAN-13 | 13 цифр | Європейський стандарт |
| EAN-8 | 8 цифр | Маленькі упаковки |
| UPC-A | 12 цифр | Американський стандарт |
| GTIN-14 | 14 цифр | Групові упаковки |
| ISBN-13 | 13 цифр | Книги |

### Валідація

Плагін автоматично перевіряє коректність номера GTIN/EAN (контрольна цифра). Некоректний номер буде відхилений з повідомленням про помилку.

### Структуровані дані (Schema.org)

Номер GTIN автоматично додається до структурованих даних продукту (JSON-LD), що покращує видимість у результатах пошуку Google:

```json
{
    "@type": "Product",
    "gtin13": "5901234123457",
    "brand": {
        "@type": "Brand",
        "name": "Nazwa marki"
    },
    "manufacturer": {
        "@type": "Organization",
        "name": "Nazwa producenta",
        "address": "ul. Przykładowa 1, 00-001 Warszawa"
    }
}
```

## Shortcode

Використовуйте shortcode `[polski_manufacturer]`, щоб відобразити дані виробника у будь-якому місці.

### Параметри

| Параметр | Тип | За замовчуванням | Опис |
|----------|-----|------------------|------|
| `product_id` | int | поточний | ID продукту |
| `fields` | string | `all` | Поля для відображення: `all`, `name`, `address`, `email`, `url`, `gtin`, `brand` |
| `layout` | string | `list` | Макет: `list`, `inline`, `table` |
| `show_label` | bool | `true` | Чи відображати мітки полів |
| `wrapper` | string | `div` | HTML-елемент обгортки |

### Приклади використання

Повні дані виробника:

```html
[polski_manufacturer]
```

Результат (макет list):

```
Producent: ABC Sp. z o.o.
Adres: ul. Fabryczna 10, 00-001 Warszawa
E-mail: kontakt@abc.pl
Strona: https://abc.pl
```

Тільки назва та GTIN:

```html
[polski_manufacturer fields="name,gtin"]
```

Бренд продукту у макеті inline:

```html
[polski_manufacturer fields="brand" layout="inline"]
```

Для конкретного продукту:

```html
[polski_manufacturer product_id="789" fields="name,address" layout="table"]
```

У шаблоні PHP:

```php
echo do_shortcode('[polski_manufacturer product_id="' . $product->get_id() . '" fields="name,gtin"]');
```

## Програмний доступ до даних

### Отримання даних виробника

```php
$manufacturer_name = get_post_meta($product_id, '_polski_manufacturer_name', true);
$manufacturer_address = get_post_meta($product_id, '_polski_manufacturer_address', true);
$manufacturer_email = get_post_meta($product_id, '_polski_manufacturer_email', true);
$manufacturer_url = get_post_meta($product_id, '_polski_manufacturer_url', true);
$gtin = get_post_meta($product_id, '_polski_gtin', true);
```

### Отримання бренду

```php
$brands = wp_get_object_terms($product_id, 'polski_brand');
if (!empty($brands) && !is_wp_error($brands)) {
    $brand_name = $brands[0]->name;
    $brand_logo = get_term_meta($brands[0]->term_id, 'thumbnail_id', true);
}
```

## Імпорт CSV

Дані виробника та GTIN можна імпортувати за допомогою CSV:

| Колонка CSV | Опис |
|-------------|------|
| `polski_manufacturer_name` | Назва виробника |
| `polski_manufacturer_address` | Адреса виробника |
| `polski_manufacturer_email` | E-mail виробника |
| `polski_manufacturer_url` | Вебсайт виробника |
| `polski_gtin` | Номер GTIN/EAN |
| `polski_brand` | Назва бренду |

Приклад:

```csv
"Krem nawilżający","ABC Kosmetyki Sp. z o.o.","ul. Kwiatowa 5, 00-100 Warszawa","info@abc.pl","https://abc.pl","5901234123457","ABC Kosmetyki"
```

## Найпоширеніші проблеми

### Дані виробника не відображаються на сторінці продукту

1. Перевірте, чи модуль виробника увімкнено
2. Переконайтеся, що продукт має заповнені дані або налаштовані дані за замовчуванням
3. Перевірте, чи тема підтримує hook `woocommerce_single_product_summary` або `woocommerce_product_meta_end`

### GTIN відхилений як некоректний

Перевірте контрольну цифру номера GTIN. Використовуйте калькулятор GS1 для перевірки: https://www.gs1.org/services/check-digit-calculator

### Бренд не з'являється у Schema.org

Переконайтеся, що бренд призначений продукту через таксономію `polski_brand`, а не просто вписаний у текстове поле виробника.
