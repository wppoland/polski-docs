## Що таке Polski for WooCommerce?

**Polski for WooCommerce** - це безкоштовний плагін з відкритим кодом (GPLv2), створений [wppoland.com](https://wppoland.com), який адаптує магазин WooCommerce до польських правових вимог та стандартів e-commerce. Плагін поєднує понад 30 модулів, що охоплюють правові вимоги, відображення цін, оформлення замовлення, харчові продукти, модулі магазину та інструменти для розробників.

Поточна версія: **1.3.2**

### Системні вимоги

Перед встановленням переконайтеся, що ваш сервер відповідає мінімальним вимогам:

| Вимога | Мінімальна версія |
|-----------|-----------------|
| WordPress | 6.4 або новіша |
| WooCommerce | 8.0 або новіша |
| PHP | 8.1 або новіша |
| MySQL | 5.7 або новіша / MariaDB 10.3+ |

:::tip[Рекомендація]
Для найкращої продуктивності рекомендуємо PHP 8.2+ та WooCommerce 9.x. Плагін регулярно тестується з останніми версіями WordPress та WooCommerce.
:::

## Огляд модулів

Плагін побудований модульно - ви вмикаєте лише ті функції, які вам потрібні. Нижче наведено опис усіх доступних груп модулів.

### Правові вимоги

Модулі, що допомагають впровадити вимоги польського та європейського законодавства:

- **GPSR (регламент безпеки продукції)** - дані виробника, імпортера та відповідальної особи на сторінках продуктів
- **Omnibus** - відображення найнижчої ціни за 30 днів до знижки, відповідно до директиви Omnibus
- **Право на відмову від договору** - форми та процедури повернення, генерація документів відмови
- **GDPR** - управління згодами, анонімізація даних, реєстр обробки
- **DSA (Акт про цифрові послуги)** - контактна точка, звітність щодо контенту, прозорість
- **KSeF** - підготовка до інтеграції з Національною системою електронних рахунків-фактур
- **Greenwashing** - перевірка та контроль екологічних заяв
- **Правові сторінки** - генерація регламенту, політики конфіденційності та політики повернень

### Ціни та інформація про продукт

Модулі, пов'язані з відображенням цін та даних продукту:

- **Ціни за одиницю** - автоматичний перерахунок та відображення цін за одиницю виміру (зл/кг, зл/л)
- **Відображення ПДВ** - інформація про ставку ПДВ та ціну нетто/брутто
- **Час доставки** - орієнтовний час виконання замовлення на сторінці продукту
- **Дані виробника** - поле виробника, бренд, каталожний номер

### Оформлення замовлення

Модулі, що змінюють сторінку оформлення замовлення та процес замовлення:

- **Кнопка замовлення** - зміна тексту кнопки на "Замовляю з обов'язком оплати" (правова вимога)
- **Правові чекбокси** - налаштовувані згоди на регламент, політику конфіденційності, розсилку
- **Пошук NIP** - автоматичне заповнення даних компанії на основі номера NIP (API GUS)
- **Подвійне підтвердження** - верифікація адреси e-mail через double opt-in

### Харчові продукти

Спеціалізовані модулі для магазинів з продуктами харчування:

- **Огляд харчових продуктів** - спеціальні поля для харчових продуктів
- **Харчова цінність** - таблиця харчової цінності відповідно до регламенту 1169/2011
- **Алергени** - виділені алергени в описі продукту (14 основних алергенів)
- **Nutri-Score** - відображення маркування Nutri-Score (A-E)

### Модулі магазину

Функції, що покращують досвід покупок:

- **Список бажань** - збереження продуктів на потім
- **Порівняння** - порівняння продуктів поруч
- **Швидкий перегляд** - перегляд продукту без покидання сторінки категорії
- **AJAX-пошук** - пошук продуктів у реальному часі
- **AJAX-фільтри** - динамічна фільтрація продуктів без перезавантаження сторінки
- **Слайдер продуктів** - каруселі продуктів з налаштовуваними параметрами
- **Значки продуктів** - етикетки типу "Новинка", "Бестселер", "Останні штуки"
- **Інші модулі** - додаткові функції магазину

### Інструменти

Модулі, що полегшують управління магазином:

- **Панель відповідності** - огляд стану правових вимог магазину в одному місці
- **Аудит сайту** - автоматична перевірка конфігурації магазину
- **Інциденти безпеки** - реєстр та управління інцидентами GDPR
- **Верифіковані відгуки** - система верифікованих відгуків клієнтів

### Для розробників

Інструменти та API для програмістів:

- **REST API** - ендпоінти для управління даними плагіна
- **Хуки (дії та фільтри)** - понад 100 хуків для розширення функціональності
- **Шорткоди** - готові шорткоди для вставки елементів у контент
- **Шаблони** - перевизначення шаблонів плагіна у темі
- **WP-CLI** - команди CLI для управління плагіном з терміналу
- **Імпорт CSV** - масовий імпорт даних продуктів
- **Блоки Gutenberg** - спеціальні блоки редактора
- **Schema.org** - автоматичні структуровані дані для продуктів

## Швидкий старт

Три кроки до працюючого магазину, що відповідає вимогам e-commerce:

1. **[Встановіть плагін](/uk/getting-started/installation/)** - з панелі WordPress або вручну з ZIP-файлу
2. **[Налаштуйте основи](/uk/getting-started/configuration/)** - увімкніть потрібні модулі в панелі налаштувань
3. **[Пройдіть майстер](/uk/getting-started/wizard/)** - заповніть дані компанії, згенеруйте правові сторінки, налаштуйте чекбокси

## Чому варто?

- **Все в одному** - замість 10 плагінів, одна цілісна платформа
- **Модульна структура** - вмикаєте лише те, що потрібно
- **Правові інструменти** - оновлюються відповідно до змін у законодавстві
- **Open source** - вихідний код на GitHub, ліцензія GPLv2
- **Без підписки** - усі функції доступні безкоштовно
- **Продуктивність** - мінімальні ресурси завантажуються лише для активних модулів
- **Активна спільнота** - підтримка на GitHub Discussions
