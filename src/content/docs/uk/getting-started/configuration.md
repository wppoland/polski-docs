## Головна панель плагіна

Після встановлення та активації плагіна перейдіть до **WooCommerce > Polski** в меню адміністративної панелі. Ви побачите головну панель (dashboard), розділену на кілька секцій:

- **Статус відповідності** - швидкий огляд, які правові вимоги виконані
- **Активні модулі** - список увімкнених модулів з посиланнями на їх налаштування
- **Необхідні дії** - сповіщення про відсутні конфігурації
- **Швидкі посилання** - посилання на найважливіші налаштування

:::tip[Майстер налаштування]
Якщо ви тільки починаєте, рекомендуємо скористатися [майстром налаштування](/uk/getting-started/wizard/), який проведе вас через найважливіші налаштування крок за кроком. Майстер можна запустити повторно в будь-який момент з панелі dashboard.
:::

## Увімкнення та вимкнення модулів

Плагін працює модульно - за замовчуванням після встановлення жоден модуль не активний. Завдяки цьому плагін не впливає на продуктивність магазину, доки ви не увімкнете конкретних функцій.

### Як увімкнути модуль

1. Перейдіть до **WooCommerce > Polski > Модулі**
2. Знайдіть модуль, що вас цікавить, у списку
3. Натисніть перемикач біля назви модуля, щоб його увімкнути
4. Натисніть **Зберегти зміни** внизу сторінки

### Як вимкнути модуль

Процедура ідентична - натисніть перемикач біля активного модуля, щоб його вимкнути. Вимкнення модуля не видаляє збережених даних, тому ви можете повторно увімкнути його без втрати конфігурації.

### Рекомендовані модулі для початку

Для типового польського інтернет-магазину рекомендуємо увімкнути наступні модулі як мінімум:

| Модуль | Чому важливий |
|-------|-------------------|
| Omnibus | Обов'язковий за законом - відображення історії цін |
| Кнопка замовлення | Обов'язковий за законом - "Замовляю з обов'язком оплати" |
| Правові чекбокси | Обов'язкові за законом - згоди при оформленні замовлення |
| Правові сторінки | Регламент та політика конфіденційності |
| Право на відмову від договору | Обов'язковий за законом - форма та інструкція про відмову |
| Час доставки | Рекомендований - орієнтовний час доставки на сторінці продукту |
| GPSR | Обов'язковий з 13.12.2024 - дані безпеки продукту |

## Панель відповідності

Панель відповідності - це центральне місце, де ви можете перевірити правовий стан свого магазину. Перейдіть до **WooCommerce > Polski > Відповідність**, щоб побачити:

### Індикатори статусу

Кожна правова вимога має один з трьох статусів:

- **Відповідає** (зелений) - вимога виконана, конфігурація завершена
- **Потребує уваги** (жовтий) - модуль увімкнений, але частина конфігурації відсутня
- **Не відповідає** (червоний) - модуль вимкнений або конфігурація неповна

### Контрольний список

Панель відображає контрольний список з конкретними кроками для виконання:

```
[x] Кнопка замовлення - текст відповідає законодавству
[x] Omnibus - відображення історії цін увімкнено
[ ] Регламент - сторінка регламенту не призначена
[ ] Політика конфіденційності - сторінка не призначена
[ ] GPSR - відсутні дані виробника на 12 продуктах
```

Натисніть будь-який елемент списку, щоб перейти безпосередньо до відповідних налаштувань.

## Налаштування окремих груп модулів

### Правова відповідність

Перейдіть до **WooCommerce > Polski > Правова відповідність**, щоб налаштувати:

**Omnibus (цінова директива)**

1. Увімкніть модуль Omnibus
2. Встановіть період відстеження цін (за замовчуванням 30 днів)
3. Виберіть формат відображення найнижчої ціни
4. Збережіть зміни

Плагін автоматично почне реєструвати історію цін з моменту активації модуля.

**GPSR (безпека продукції)**

1. Увімкніть модуль GPSR
2. Заповніть дані виробника за замовчуванням у глобальних налаштуваннях
3. Для окремих продуктів - редагуйте дані у вкладці "GPSR" на сторінці редагування продукту

**Правові сторінки**

1. Увімкніть модуль правових сторінок
2. Скористайтеся генератором для створення регламенту, політики конфіденційності та політики повернень
3. Призначте згенеровані сторінки у **WooCommerce > Налаштування > Розширені > Налаштування сторінки**

### Ціни та інформація про продукт

Перейдіть до **WooCommerce > Polski > Ціни**, щоб налаштувати:

**Ціни за одиницю**

1. Увімкніть модуль цін за одиницю
2. Виберіть одиницю виміру за замовчуванням (кг, л, м, шт.)
3. На сторінці продукту заповніть поле "Базова кількість" та "Одиниця виміру"

Приклад налаштування в редакторі продукту:

```
Ціна продукту: 15,99 зл
Базова кількість: 500
Одиниця виміру: г
Референсна одиниця: кг

Результат: 15,99 зл / 500г (31,98 зл/кг)
```

**Час доставки**

1. Увімкніть модуль часу доставки
2. Встановіть час доставки за замовчуванням (наприклад, "1-3 робочі дні")
3. Опціонально - встановіть індивідуальний час для окремих продуктів

### Оформлення замовлення

Перейдіть до **WooCommerce > Polski > Каса**, щоб налаштувати:

**Кнопка замовлення**

1. Увімкніть модуль
2. Текст за замовчуванням - "Zamawiam z obowiązkiem zapłaty"
3. Ви можете налаштувати текст, але він повинен відповідати вимогам ст. 17 закону про права споживачів

**Правові чекбокси**

1. Увімкніть модуль чекбоксів
2. Додайте необхідні згоди (регламент, політика конфіденційності)
3. Налаштуйте текст кожного чекбоксу, включаючи посилання на правові сторінки
4. Позначте, які чекбокси є обов'язковими

Приклад налаштування чекбоксу:

```
Мітка: регламент
Текст: Ознайомився/лася з [регламентом] і приймаю його положення.
Обов'язковий: Так
Посилання: /regulamin/
Позиція: Перед кнопкою замовлення
```

**Пошук NIP**

1. Увімкніть модуль NIP
2. Поле NIP з'явиться автоматично на сторінці оформлення замовлення
3. Після введення NIP та натискання "Перевірити" дані компанії заповняться автоматично з бази GUS

### Харчові продукти

Ці модулі призначені для магазинів, що продають продукти харчування. Перейдіть до **WooCommerce > Polski > Харчування**.

1. Увімкніть потрібні модулі (харчова цінність, алергени, Nutri-Score)
2. На сторінці редагування продукту з'являться нові вкладки для заповнення даних
3. Дані відобразяться автоматично на сторінці продукту в магазині

### Модулі магазину

Перейдіть до **WooCommerce > Polski > Магазин**, щоб увімкнути додаткові функції:

- Список бажань, порівняння, швидкий перегляд - увімкніть та налаштуйте зовнішній вигляд
- AJAX-пошук - увімкніть та налаштуйте кількість відображуваних результатів
- AJAX-фільтри - увімкніть та виберіть атрибути для фільтрації
- Слайдер та значки - налаштуйте стиль та поведінку

## Глобальні налаштування

У вкладці **WooCommerce > Polski > Налаштування** знайдете глобальні опції:

### Дані компанії

Заповніть основні дані вашої компанії:

- Назва компанії
- NIP
- REGON
- Адреса офісу
- Контактна адреса e-mail
- Номер телефону

Ці дані використовуються різними модулями (правові сторінки, GPSR, DSA).

### Продуктивність

- **Завантаження ресурсів** - CSS та JS завантажуються лише на сторінках, де вони потрібні
- **Кеш** - плагін використовує Transients API WordPress для кешування даних
- **Мініфікація** - ресурси фронтенду мініфіковані

### Сумісність

Якщо ви використовуєте нестандартну тему або плагіни, що спричиняють конфлікти:

1. Перейдіть до **WooCommerce > Polski > Налаштування > Сумісність**
2. Увімкніть режим сумісності для проблемних модулів
3. Налаштуйте пріоритети хуків, якщо елементи відображаються у неправильному порядку

## Перевірка конфігурації

Після налаштування модулів варто перевірити, чи все працює:

1. **Панель відповідності** - перейдіть до **WooCommerce > Polski > Відповідність** і перевірте, чи всі індикатори зелені
2. **Сторінка продукту** - відкрийте будь-який продукт у магазині та перевірте, чи відображаються нові елементи (ціна Omnibus, час доставки, дані GPSR)
3. **Сторінка оформлення** - зробіть тестове замовлення та перевірте, чи чекбокси та кнопка правильні
4. **Правові сторінки** - відкрийте регламент та політику конфіденційності та перевірте їх зміст

Також можна запустити автоматичний аудит: **WooCommerce > Polski > Інструменти > Аудит сайту**.
